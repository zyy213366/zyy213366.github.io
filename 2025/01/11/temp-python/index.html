例题
假设我们有一张钢材信息表，包含以下字段：

steel_id：钢材编号
steelspec：钢种
slabweight：钢材重量
slabthickness：钢材厚度

目标：

  1. 重命名 steel_id 为 steel_code。
  2. 找出 slabweight 最大值对应的钢材信息。
  3. 按 slabweight 降序排列钢材信息表。
  4. 筛选出钢种为 X50M 的钢材，并计算其平均重量。
  5. 筛选出钢种以 X 开头的钢材，将其厚度加 400，并将结果合并回主表。
  6. 按钢种分组，计算每种钢材的平均重量，并命名结果列为 mean_slabweight。
